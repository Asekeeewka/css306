## WAF - Web Application Firewall
A WAF creates a shield between a web app and the Internet; this shield can help mitigate many common attacks.(cloudflare)

1. Free WAF
   - Nginx + nasxi
   - Apache + modesecurity
2. Proprietary + expensive
   - Imperva WAF
   - Nemeside AI
   - PT AF
3. Proprietary + not expensive
   - Cloudflare WAF
